Instructions:
1. Paste the files inside a folder in your computer.
2. Press and hold the Shift button then right click on an empty space in the folder.
3. Click on "Open command window here".
4. Type "mvn clean install" and press enter. Type "exit" and press enter.
5. A target folder will be generated inside the app folder.
6. Open the target inside the app folder.
7. Press and hold the Shift button then right click on an empty space in the target folder.
8. Click on "Open command window here".
9. Type "java -jar app-1.0.0.jar" and press enter to run the JAR file or you can also 
indicate the name of the text file after the JAR file name and a space 
(i.e. java -jar app-1.0.0.jar file.txt).
